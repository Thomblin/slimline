Class Request_Filter_Controller

Description | Vars (details)

Request_Filter is used to store data definitions according to filter_var() Each allowed input variable should be stored as public variable

e.g.

public $name = array( 'scope' => array('GET', 'POST', 'COOKIE', 'SERVER', 'CLI', 'USER'), 'filter' => FILTER_DEFAULT, 'options' => array('default' => 'sebastian'), );
